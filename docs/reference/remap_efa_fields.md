# Remap EFA field names to the convention efourier\_i expects

<div class="ref-description section level2">

Remap EFA field names to the convention efourier\_i expects

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
remap_efa_fields(ef)
```

</div>

</div>

<div class="section level2">

## Arguments

-   ef:

    A list returned by efourier or efourier\_norm

</div>

<div class="section level2">

## Value

The same list with fields named an, bn, cn, dn, a0, c0

</div>
